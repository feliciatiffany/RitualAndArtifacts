# Images Assets

This folder contains visual assets for the Elevator Bet Game.

## Current Files (SVG placeholders)

- `elevator-1.svg` – Blue elevator (E1)
- `elevator-2.svg` – Red elevator (E2)
- `elevator-3.svg` – Yellow elevator (E3)
- `elevator-4.svg` – Teal elevator (E4)
- `transition-1.svg` – Transition screen, Blue
- `transition-2.svg` – Transition screen, Red
- `transition-3.svg` – Transition screen, Yellow
- `transition-4.svg` – Transition screen, Teal

## Optional Replacements

You can replace the SVG placeholders with custom PNG/JPG assets for a more polished look:

- Comic-style elevator door illustrations
- Custom transition burst graphics
- Background images for start/thanks screens (referenced in CSS)

## Ranking QR

The RANKING popup now uses a **dynamically generated** QR code (via QRCode.js) pointing to `stats.html`. No static `ranking-qr.png` is required.
